# 🌳 smarTree — Academic Job Tree

분야의 가지를 따라 내려가면, 내 공고가 보입니다.

학술 채용공고(교수·연구원·박사후연구원)를 **인터랙티브 분야 트리**로 탐색하는
사이트 프로토타입입니다. 하이브레인넷 등 기존 학술 채용 게시판의 목록 중심
탐색을 벤치마킹하되, 분야 구조·스킬·시간(아카이브) 축을 시각적으로 재구성했습니다.

**라이브 데모**: GitHub Pages 활성화 후 → `https://ckim-test.github.io/smarTree/`

## 주요 기능

- **버블 트리 탐색** — 분야별 공고 수가 원 안에 실시간 표시. 호버 시 확대,
  드래그 가능(스프링 물리), 여러 분야 동시 선택(합집합 필터)
- **스킬 3단계 필터** — 스킬 칩 클릭마다 `+포함 → −제외 → 해제` 순환.
  제외는 필수 요건에만 적용 (우대 사항은 없어도 지원 가능하므로)
- **공고 나란히 비교** — 최대 3건, 필수·우대 스킬 합집합 매트릭스로 비교
- **연도별 아카이브 드릴다운** — 마감 공고는 삭제 대신 보존.
  아카이브 → 연도 → 분야 순으로 트리가 펼쳐짐
- **PDF 자동 등록 (데모)** — 대학 일괄 초빙공고 PDF를 Claude API로
  구조화 추출: 모집단위별 분리, 분야 자동 분류, 스킬 태깅, 확신도 기반 검수 분기
- **수목원 테마** — 라이트/다크 모드 (헤더 🌙 버튼, 시스템 설정 자동 추종)

## 실행

정적 단일 파일입니다. `index.html`을 브라우저로 열면 끝.

```sh
open index.html          # 또는
python3 -m http.server   # http://localhost:8000
```

## 배포 (GitHub Pages)

저장소 **Settings → Pages → Source: Deploy from a branch → `main` / `(root)`** 설정.
1~2분 후 `https://<아이디>.github.io/smarTree/` 에서 접속 가능.

## 구조

```
index.html                  # 사이트 전체 (HTML+CSS+JS 단일 파일, 목데이터 포함)
pipeline/extract_postings.py # 공고 PDF → 구조화 추출 스크립트 (Claude API)
```

### PDF 추출 파이프라인 (`pipeline/`)

```sh
pip install anthropic pydantic
export ANTHROPIC_API_KEY=...          # ⚠️ 키는 서버/로컬에만 — 클라이언트 코드에 넣지 말 것
python pipeline/extract_postings.py 공고문.pdf > postings.json
```

분야 분류는 사이트 트리 택소노미(enum)로 강제되어 후처리 매핑이 필요 없습니다.
대량 처리는 Batches API로 옮기면 비용 50% 절감.
